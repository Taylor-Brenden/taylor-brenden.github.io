Arrays

An array is a variable with multiple values stored inside.

You store an array like this: var cars = ["Ford", "Chevy", "BMW"];

After this array has been stored you can call the values by their index: 0,1, or 2.

We will use this code to call the array:
function sayHello() {
var cars = ["Ford", "Chevy", "BMW"];
var index = document.getElementById("index").value;

document.write (cars[index]);
}

Pick one index to call the value of the car array
