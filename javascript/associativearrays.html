Associative Arrays

An associative array is a dynamic object that is like an array but it can have keys assigned to its values.

You create an associative array like this: var arr = { "one": 1, "two": 2, "three": 3 };

Values are called like this: arr["one"];

We will use a for in loop to list out the values of an associative array:

I created an associative array with the make of the car as the key and the model to be the value:
var arr = {"Ford": "Fusion", "Chevy": "Malibu", "BMW": "M5"};
for (var key in arr)
{
var value = arr[key];
document.write(key + " = " + value + '<br>');
}
