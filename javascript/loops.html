While Loop

A simple while loop that adds 1 to the variable "count" as long as it is less than 10.

var count = 0;
document.write("Starting Loop ");

while (count < 10) {
document.write("Current Count : " + count + "<br />");
count++;
}

document.write("Loop stopped!");
